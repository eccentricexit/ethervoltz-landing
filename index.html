Liquid democracies on the blockchain.

EtherVotes is a distributed, serverless application that guarantees zero-down time, DDoS resistance and blockchain security against election frauds.

Centralized democracies are not democratic.

Technology has evolved tremendously with the rise of the internet. Governance has not. This software is a distributed application that aims to implement liquid democracy in a way that guarantees zero-downtime, DDoS resistance and fraud resistance.

Wouldn't it be nice to vote on presidential elections from your smartphone? Well, centralized databases for storing election votes are extremely valuable for hackers and people attempting election frauds.

This is why a decentralized, cryptographically secured solution is the way.

Visit the project page.

Serverless

There is no IP for attacker to target. Files are stored on Swarm. This very page was served without a centralized server.

Fraudless

Fraudless. Blockchain technology guarantees that no one can issue votes out of thin air.
