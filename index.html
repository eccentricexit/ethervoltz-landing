Liquid democracies on the blockchain.

Current governance is broken.

Technology has evolved tremendously with the rise of the internet. Governance has not. This software is a distributed application that aims to implement liquid democracy in a way that guarantees zero-downtime, DDoS resistance and fraud resistance.

Wouldn't it be nice to vote on presidential elections from your smartphone? Well, centralized databases for storing election votes are extremely valuable for hackers and people attempting election frauds.

Enter EtherVotes

The ethereum virtual machine (EVM) together with Swarm allow the construction of serverless distributed applications.

Distributed Application

The app (and this website) are hosted on Swarm, allowing for zero downtime and DDoS resistance.

Distributed database

Issues, vote delegation and all relavant information are safely stored on the ethereum blockchain.

Open Source

The source code is available for auditing and contribution on Github.
